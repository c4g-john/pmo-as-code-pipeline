## Overview

User stories for the self-serve onboarding flow, each tracing to a product
requirement.

## User Stories

- **AUR-US-001** (traces: AUR-PR-014): As a new customer, I want to complete account setup myself, so that I can start using the product on my first day.
- **AUR-US-002** (traces: AUR-PR-015): As a customer part-way through onboarding, I want progress updates by email, so that I always know what is left to do.
